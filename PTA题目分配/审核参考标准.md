# 审核参考标准

> 对于不规范的 issue 请予以评论指出问题，而后微信通知该同学及时修改，没问题的issue及时close，同时也微信通知该同学上传微云

## 文字内容

| 问题类型 | 说明 |
|---------|------|
| 格式臃肿 | "实验报告"样式仍繁杂 |
| 描述不清 | 问题描述模糊、存在逻辑错误 |

## 测试集部分

| 问题类型 | 说明 |
|---------|------|
| 内容错误 | 代码块中文字内容不正确 |
| 位置错误 | 设计说明写在代码块内 |
| 不符合规范 | 测试集格式/结构不达标 |

## 代码部分

| 问题类型 | 说明 |
|---------|------|
| 测试不通过 | 代码无法通过测试集 |
| 测试有问题 | 测试集本身存在缺陷 |
