# 算法设计与分析

> 本仓库用于课程《算法设计与分析》的学习交流与问题反馈。

## 如何提交 Issue

### 提交题目报错
如果你发现作业题描述、数据或答案有误，请使用 **📝 作业题报错** 模板提交 Issue。

### 提交课程建议
如果你对课程内容、作业安排或仓库维护有建议，请使用 **💡 课程建议/反馈** 模板提交 Issue。

### 其他问题/讨论
有其他问题或想发起讨论，请使用 **❓ 其他问题/讨论** 模板提交 Issue。

> 点击仓库上方的 **Issues** 标签页，然后点击 **New Issue** 即可选择模板提交。
